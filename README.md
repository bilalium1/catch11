# Catch11 - implementation of catch11 on the terminal

### ! - This game is only compatible with UNIX-based systems

## How to build (yes you will need to build this game.)

### Install dependencies :

#### Arch Linux :
```bash
sudo pacman -S base-devel
```

#### Debian/Ubuntu :
```bash
sudo apt install build-essential
```

### Build Game : 
- to build this game, run : 
```bash
make
```
- now that you have the binary file, you can run it directly or run it with : 
```bash
make run
```
